# own-your-style
Own Your Style is a Flutter mobile application that digitizes a userâ€™s wardrobe by extracting clothing attributes using computer vision. Outfit selection is modeled as a constrained optimization problem and solved with a genetic algorithm that generates and ranks outfit combinations based on weather, location, and event type. The app pairs a polished Flutter frontend with a Laravel backend service to store wardrobe data and deliver personalized recommendations.
